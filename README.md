# Documents N Dice, a physics based dice roller and character sheet manager!
Welcome to Documents N Dice. This is a piece of open source software meant to help organise online DND sessions made in Unity. Features include:
 - Multiplayer to roll dice together
 - Dice face customiser to easily add modifiers or roll odd dice
 - Whiteboards and notepads to draw up strategies and plans or write character or setting notes
 - DM-Only area for secret notes
 - Chatlog to send messages to each player

# How to use
Installing this software is simple, simply download the latest software and run it. Using as a client is also simple. Select client settings, input the correct IP and port, and you are off to the races.  
Using this as a host is slightly different. First, you MUST port forward. There are plenty guides on how to do this, but if you are confused ChatGPT or similar will be able to help you. Once you do that, Select Host settings, input the desired port (I reccomend 7776). Send your ip and port to the clients you want to join and await their imminent arrival.

# How to use in Unity
To use this in Unity, you will need to download the contents of the repo and place in the Assets folder. Additionally, install TextMeshPro package if not already installed. Once done, you are good to go.
